1. As a Visitor, I want to enter ingredients I have, so that I can get a complete meal idea fast.
2. As a Visitor, I want to search by a dish name, so that I can generate a full recipe on demand.

5. As a Cook, I want to save a pantry profile and allergens, so that generated recipes respect my constraints.
6. As a Cook, I want to save recipes, so that I can find and reuse my favorites later.
7. As a Cook, I want to rate recipes and add personal notes, so that I can remember adjustments for next time.
8. As a Cook, I want to export a shopping list from a recipe, so that I can buy missing items efficiently.
9. As a Cook, I want a step-by-step cook mode with timers, so that I can follow instructions hands-free while cooking.
10. As a Cook, I want estimated nutrition and macros per serving, so that I can meet my dietary goals.
11. As an Admin, I want to monitor basic usage metrics, so that I can plan capacity and detect issues early.
12. As an Admin, I want to flag and hide abusive prompts or outputs, so that I can keep the platform safe for users.
